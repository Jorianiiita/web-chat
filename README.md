# Web Chat
A Chat application similar to whats app web.

## Setup
1. Clone the repository `git@github.com:Jorianiiita/web-chat.git`.
2. Install npm and node latest version.
3. Run npm install.
```sh
$ npm install
$ cd src/server
$ npm install
```
4. Run npm run start.
```sh
$ npm run start
```
5. Goto `http://localhost:8001` to see web-chat.
